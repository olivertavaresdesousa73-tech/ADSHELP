🔒 Política de Privacidade

Última atualização: 2025

1. Dados Coletados

A ADHELP não coleta nem transmite dados pessoais. Todas as informações (histórico, salvos, configurações) ficam armazenadas exclusivamente no localStorage do seu navegador.

2. API da Meta

As consultas à Meta Ads Library incluem apenas os termos de busca e o país selecionado. Nenhum dado pessoal é transmitido.

3. Exclusão de Dados

Para remover todos os dados: Configurações → Resetar Sistema.

📄 Termos de Uso

Última atualização: 2025

1. Uso Aceitável

  • Pesquisa e análise de concorrência legítima
  • Planejamento de campanhas próprias
  • Geração de estratégias para uso próprio

2. Uso Proibido

  • Copiar criativos sem autorização
  • Uso para fins ilegais ou enganosos

3. Isenção

A ADHELP é fornecida "como está". Não nos responsabilizamos por decisões baseadas nas análises da plataforma.
